## ⚠️ **What You Must Do (Copyleft Requirements)**

### **Source Code Availability**
If you distribute Keyorix or run a modified version on a server that others can access:

1. **Provide Source Code**: You must make the complete source code available
2. **Include License**: You must include the AGPL-3.0 license
3. **Network Copyleft**: If users interact with your modified version over a network, you must provide them access to the source code
4. **Same License**: Your modifications must also be licensed under AGPL-3.0

### **Key AGPL-3.0 Requirement: Network Copyleft**
Unlike GPL, AGPL-3.0 has a "network copyleft" provision:

> If you run a modified version of Keyorix on a server and let others interact with it remotely through a computer network, you must provide those users access to the source code of your modified version.

## 🔧 **Technical Compliance**

### **Source Code Disclosure Methods**
When required to provide source code, you can:

1. **Direct Download**: Provide a download link to the complete source
2. **Version Control**: Provide access to a Git repository
3. **Archive**: Provide a compressed archive of the source code
4. **Physical Media**: Provide source code on physical media (if distributing physically)

### **What Must Be Included**
